OPDRACHT GELUKT OPMERKING
Minimaal - -
1.1 – Koppen Ja Nooit meer dan een h1 tag. Voor de rest, geen opmerkingen.
1.2 – Afbeelding Ja Gallery-page: een responsieve image-grid van verschillende konijnenrassen. Bij een cursor hover op elke afbeelding wordt de afbeelding groter contrast en helderheid +0.2
1.3 – Link Ja Links in de navigatiebar, footer, afbeeldingen, knoppen.
1.4 – Afbeelding link Ja De afbeeldingen op de gallery-page nemen je naar de Wikipedia pagina van het konijnenras waar je op geklikt hebt.
1.5 – Tabel Ja Deze pagina -- geen extra opmerkingen.
1.6 – Lijst Ja De footer onderaan elke pagina bevat (ul) lijsten.
1.7 – Meta Ja In de header van elke pagina bevatten meerdere meta tags, zoals description, favicon, viewport etc.
2 – 3 pagina's Ja In totaal 4 pagina's (excl. BunnyGreenies) -- geen extra opmerkingen.
3 – Navigatie Ja Elke pagina heeft een responsieve bootstrap navbar bovenaan.
4 – Externe stylesheet Ja In plaats van een css bestand voor alle pagina's, heb ik ervoor gekozen om het op te delen: dus voor elke html pagina een css bestand, voor gemak. Als ik aan deze website verder zou werken zou ik kiezen voor een main css bestand als de basis voor alle html pagina's.
5 – Kleur gebruik Ja Deze website bevat lichte kleuren, vooral met rgb(a).
6 – Validatie Ja Geen extra opmerkingen.
7 – Online beschikbaar Ja Ik host deze website met Heroku.com, en bij elke (grote) wijziging commit & push ik de wijzigingen via de Windows terminal met git version control.
Extra - -
8 – Knop Ja De rode knop op de index pagina leidt naar een externe site, BunnyGreenies, dat niet door mij is gemaakt maar door een vriend. De rede dat ik het op mijn website heb toegevoegd is omdat beide websites over konijnen gaan.
9 – Video Ja Gallery-pagina, video van springende konijnen!
10 – Contact form Ja Contact form gemaakt met bootstrap, autofocus op "First name". Hoewel het mogelijk is voor mij om server-side te coderen doordat ik Heroku gebruik en niet Github (wat alleen static websites toelaat voor gratis gebruikers), heb ik er niet voor gekozen omdat het dan te veel tijd zou kosten. Als ik het zou doen, zou ik het via een Python framework zoals Flask coderen.
11 – Verras docent Ja De toolbars op de index pagina en de BunnyGreenies, HTML/CSS PO links in de navbar. Verder; de footer, de image grid (+ het vergroten van de afbeeldingen) op de gallery-page, de favicon.
12 – Extra uitleg - Ik heb aan de ene kant veel geleerd over web development, en aan de andere kant mijn al bestaande vaardigheden verbeterd.
Waar ik het meest trots op ben is de image grid en dezelfde 'thema' (kleuren, opbouw) van de html pagina's.
De reden dat ik voor het onderwerp konijnen gekozen heb is simpel: mijn vriend, 'Nyandereian' die de BunnyGreenies website heeft ontworpen, heeft zelf een konijn: dus dacht ik om het maar over konijnen te hebben.
